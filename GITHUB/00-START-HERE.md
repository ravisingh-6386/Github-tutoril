## 💡 **Quick Start**

If you just want to get started NOW:

```bash
# 1. Install Git (see 02-Installing-Git.md)

# 2. Configure Git
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# 3. Create a folder for your project
mkdir my-project
cd my-project

# 4. Initialize Git
git init

# 5. Create a file
echo "Hello GitHub!" > README.md

# 6. Add and commit
git add README.md
git commit -m "Initial commit"

# 7. Create repo on GitHub.com, then:
git remote add origin https://github.com/yourusername/my-project.git
git branch -M main
git push -u origin main
```

## 🔑 **Key Concepts Overview**

| Concept | What It Is |
|---------|-----------|
| **Repository** | Folder containing your project + history |
| **Commit** | Snapshot of code changes |
| **Branch** | Independent line of development |
| **Remote** | GitHub copy of your repo |
| **Pull Request** | Proposal to merge changes |
| **Fork** | Your own copy of someone's repo |
| **Issue** | Bug or feature request tracker |
